## 部署方案

### 方案一：GitHub Pages（免费，推荐）

1. 注册/登录 [GitHub](https://github.com)
2. 创建新仓库，命名为 `faast-website`
3. 上传所有文件到仓库
4. 进入 Settings → Pages
5. Source 选择 "Deploy from a branch"
6. Branch 选择 "main"，文件夹选择 "/ (root)"
7. 保存后等待几分钟，访问 `https://你的用户名.github.io/faast-website/`

### 方案二：Netlify（免费）

1. 访问 [Netlify](https://www.netlify.com)
2. 注册/登录账号
3. 点击 "Add new site" → "Deploy manually"
4. 将整个项目文件夹拖放到上传区域
5. 等待部署完成，获得网站链接
6. 可在 Settings → Domain management 中设置自定义域名

### 方案三：Vercel（免费）

1. 访问 [Vercel](https://vercel.com)
2. 使用 GitHub 账号登录
3. 点击 "Add New Project"
4. 导入 GitHub 仓库或直接上传文件
5. 自动部署，获得 `.vercel.app` 域名

### 方案四：学校服务器

联系厦门大学网络中心或相关部门，申请：
- 服务器空间
- 域名（如 `faast.xmu.edu.cn`）
- FTP/SFTP 上传权限

## 部署前修改

部署前请将 `index.html` 中的以下内容替换为实际信息：

```html
<!-- 第9行 -->
<link rel="canonical" href="https://your-domain.com/">
```

替换为实际域名，如：
```html
<link rel="canonical" href="https://faast.xmu.edu.cn/">
```

## SEO优化说明

网页已添加以下SEO元素：

1. **Meta标签**：标题、描述、关键词、作者
2. **结构化数据**：Schema.org 标记，帮助搜索引擎理解内容
3. **语义化HTML**：使用正确的标题层级和标签
4. **响应式设计**：适配移动端和桌面端

## 搜索引擎收录

部署后提交到搜索引擎：

### 百度收录
1. 访问 [百度搜索资源平台](https://ziyuan.baidu.com)
2. 添加网站并验证所有权
3. 提交 sitemap 或单个链接

### Google收录
1. 访问 [Google Search Console](https://search.google.com/search-console)
2. 添加属性并验证
3. 提交站点地图

### Bing收录
1. 访问 [Bing Webmaster Tools](https://www.bing.com/webmasters)
2. 添加网站并验证
3. 提交URL

## 注意事项

1. **域名备案**：如使用国内服务器，需进行ICP备案
2. **HTTPS**：建议使用HTTPS协议，提升安全性和搜索排名
3. **定期更新**：保持内容更新有助于搜索引擎抓取
4. **访问统计**：可添加百度统计或Google Analytics追踪访问量
