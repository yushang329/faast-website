# FaAST 网页快速部署步骤

## 方案：GitHub Pages（推荐，免费）

### 第一步：准备文件

所有文件已准备就绪，都在 `D:\XMU\Subject\互联网＋\项目网页` 文件夹中：
- index.html
- logo.png
- 所有团队成员照片（.png）
- 所有顾问照片（.png）

### 第二步：创建 GitHub 账号

1. 访问 https://github.com
2. 点击 "Sign up" 注册账号（使用邮箱）
3. 完成邮箱验证

### 第三步：创建仓库

1. 登录 GitHub
2. 点击右上角 "+" → "New repository"
3. 仓库名称填写：`faast-website`
4. 选择 "Public"（公开）
5. 勾选 "Add a README file"
6. 点击 "Create repository"

### 第四步：上传文件

#### 方法A：网页上传（简单）

1. 进入刚创建的仓库页面
2. 点击 "Add file" → "Upload files"
3. 将项目网页文件夹中的所有文件（15个）拖放到上传区域
4. 点击 "Commit changes"

#### 方法B：Git命令行（需要安装Git）

```bash
# 安装 Git：https://git-scm.com/download/win

# 打开命令提示符，进入项目文件夹
cd "D:\XMU\Subject\互联网＋\项目网页"

# 初始化仓库
git init

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit"

# 连接远程仓库（替换为你的用户名）
git remote add origin https://github.com/你的用户名/faast-website.git

# 推送
git branch -M main
git push -u origin main
```

### 第五步：启用 GitHub Pages

1. 在仓库页面，点击 "Settings"（设置）
2. 左侧菜单点击 "Pages"
3. "Source" 部分选择 "Deploy from a branch"
4. "Branch" 选择 "main"，文件夹选择 "/ (root)"
5. 点击 "Save"
6. 等待 1-2 分钟，页面会显示网站链接：`https://你的用户名.github.io/faast-website/`

### 第六步：验证网站

1. 打开浏览器，访问显示的链接
2. 确认网页正常显示
3. 检查所有图片是否加载

### 第七步：提交搜索引擎（让更多人搜到）

## 常见问题

**Q: 图片显示不出来？**
A: 确保所有 .png 文件都已上传到仓库根目录

**Q: 网页样式错乱？**
A: 清除浏览器缓存（Ctrl+Shift+R 强制刷新）

**Q: 想使用自定义域名？**
A: 在仓库 Settings → Pages → Custom domain 中设置（需要购买域名并配置DNS）
